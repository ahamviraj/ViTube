## ✨ Overview

ViTube is a privacy-first media platform inspired by YouTube, designed exclusively for local media.

Instead of requesting access to your entire storage, ViTube only reads the folder you explicitly choose using Android's Storage Access Framework (SAF).

No cloud.

No ads.

No tracking.

Just your media.
